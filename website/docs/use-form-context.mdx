---
title: useFormContext()
---

`useFormContext` is a hook that will return the FormContext. It is meant to be used when you need to use form methods in
a deeply nested component.

:::info Provider required
[`FormProvider`](form-provider) is required in order to use this hook
:::

:::note Note
If you just need field manipulation use [`useField`](use-field) or [`useArrayField`](use-array-field) instead.
:::

```ts
function useFormContext(): FormContext;
```

## FormContext

The form context created by [`useForm`](use-form).
