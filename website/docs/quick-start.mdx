React Next Form is a lightweight React library for form state management. It eliminates a lot of boilerplate code, and less code means fewer bugs and things to worry about.

## Installation

React Next Form relies on [immer](https://immerjs.github.io/immer/docs/introduction) for immutable updates and [yup](https://github.com/jquense/yup) for schema validation.

```bash
# npm
npm i --save react-next-form immer yup

# or yarn
yarn add -E react-next-form immer yup
```

## Basic form

```jsx
import React from "react";
import { useForm } from "react-next-form";

export default function App() {
  const form = useForm({
    initialValues: {
      name: "",
    },
    onSubmit: async (values) => {
      alert(`Hey ${values.name?.length > 0 ? values.name : "buddy"}`);
    },
  });

  return (
    <div className="App">
      <h1>Hello {form.values.name.length > 0 ? form.values.name : "buddy"}!</h1>
      <form onSubmit={() => form.submit()}>
        <div className="input">
          <label htmlFor="foo">Your name:</label>
          <input
            id="foo"
            placeholder="type your name"
            type="text"
            {...form.fieldProps("name")}
          />
        </div>
        <button type="submit">Submit</button>
      </form>
    </div>
  );
}
```
